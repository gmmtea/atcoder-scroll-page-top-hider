# AtCoder Scroll Page Top Hider

[![Version](https://img.shields.io/badge/version-v1.0.1-blue)]()

AtCoder の問題ページ左下に表示される「↑ページトップ」リンクを非表示にします。

## 機能

ページ読み込み後に `#scroll-page-top` 要素を自動で削除
